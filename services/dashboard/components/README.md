# components/

Shared React components for the dashboard.

| Component | Description |
|---|---|
| `LogStream.tsx` | Connects to the SSE endpoint and renders live build logs |
| `DeployButton.tsx` | Triggers a new deployment via the API |
| `DomainManager.tsx` | UI for adding, verifying, and removing custom domains |
| `DeploymentStatus.tsx` | Displays deployment status badge and elapsed time |
